# 🎬 AI Netflix Recommender

An AI-powered movie and TV recommendation system built using **LangChain, RAG, ChromaDB, Hugging Face Embeddings, Mistral AI, Streamlit, and OMDb API**.

Unlike traditional recommendation systems, this project understands natural language queries such as:

- "Suggest horror movies from the 2010s."
- "Recommend emotional sci-fi movies."
- "Action movies starring Tom Cruise."
- "Funny family movies for the weekend."

The application retrieves relevant titles using semantic search and generates personalized recommendations with the help of a Large Language Model.

# ⚙️ How It Works

## Step 1

The user enters a natural language query.

Example:

```
Suggest horror movies from the 2010s
```

↓

## Step 2

The Query Analyzer extracts useful information such as:

- Genre
- Actor
- Director
- Language
- Release Year
- Mood
- Keywords

↓

## Step 3

Relevant movies are retrieved from the Vector Database using semantic search.

↓

## Step 4

The retrieved context is passed to Mistral AI.

↓

## Step 5

The LLM generates personalized recommendations.

↓

## Step 6

Movie posters are fetched from the OMDb API and displayed in the Streamlit interface.
